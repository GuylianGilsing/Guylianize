Welcome

Thank you for visiting this website about this framework. Guylianize is a very small framework, a micro-framework if you will. The goal of the framework is to give developers some base styling and provide some basic layouts.

Installing

You have 2 options of including Guylianize inside your project. Option 1 would be to download Guylianize from the cdn. Option 2 would be to include the code below:

<link rel="stylesheet" href="https://cdn.guyliangilsing.me/frameworks/guylianize/v1.0/guylianize.min.css" type="text/css">

The code above will load the css file as an external stylesheet. It is highly recommended that you download Guylianize through the cdn. That way you will know for sure that the framework will work within your project.

General information

To be able to make full use of this framework, some general knowledge on CSS and responsive web design (RWD) is needed. Classes are being prefixed with a . (dot), DO NOT USE THIS DOT WHEN APPLYING A CLASS (It will not work)

Typography

Guylianize will handle the font-family and font-size automatically for you. However, it does give you classes that you can apply to textual elements. The goal of these classes is to create an easy way of creating quick pieces of text that will fit your needs.

Font size and weight

You can use the following classes to alter the size and weight of a text element:

  • .h1 (H1 styling)
  • .h2 (H2 styling)
  • .h3 (H3 styling)
  • .h4 (H4 styling)
  • .h5 (H5 styling)
  • .h6 (H6 styling)
  • .font-thin (font-weight 100)
  • .font-extralight (font-weight 200)
  • .font-light (font-weight 300)
  • .font-normal (font-weight 400)
  • .font-medium (font-weight 500)
  • .font-semibold (font-weight 600)
  • .font-bold (font-weight 700)
  • .font-extrabold (font-weight 800)
  • .font-black (font-weight 900)

Color

Changing the text color is quite simple. All you need to do is add the .txt-color-YOURCOLORNAME class to the specific text element. By using the .txt-color-light and .txt-color-dark you can make the shade lighter or darker. You can find the names of the color classes and their shades in the Colors section.

Working with colors

Working with colors is quite simple. If you look at the color summary, you can see that there is a line that states "PREFIX-color-red". The prefix part of this text will tell Guylianize which styling to apply.

The following prefixes are available:

  • .bg (Background color)
  • .txt (Text color)
  • .btn (Button color)

You are not bound by these specific shades. In order to make the shade darker or lighter you can use the following classes:

  • .PREFIX-color-light
  • .PREFIX-color-dark

Layouts

It is very imporant that developers can easily create layouts. Guylianize comes with a few classes and systems to make the creation of layouts a bit faster and easier.

Procentual

Guylianize uses a procentual column layout for horizontal layouts. Instead of the commonly used 12 column layout, Guylianize will allow you to specify the width of the columns in percentages for a more design oriented control.

To start using a procentual layout, you need to have an element with the .procentual-layout class

<div class="procentual-layout">
    <div class="col-1-3 col-m-50 col-s-100 col-xs-100"></div>
    <div class="col-1-3 col-m-50 col-s-100 col-xs-100"></div>
    <div class="col-1-3 col-m-50 col-s-100 col-xs-100"></div>
</div>
